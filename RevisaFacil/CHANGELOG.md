# Changelog - RevisaFácil

## [1.1.0] - 2026-04-12
### Adicionado
- Integração automática entre "Meus Assuntos" e o Calendário.
- Sistema de proteção de anotações manuais (AssuntoId -2).
- Notificações diárias via Telegram com resumo de revisões.
- Agrupamento de múltiplos assuntos no mesmo card do calendário.

### Corrigido
- Erro que apagava anotações manuais ao sincronizar revisões.
- Problema visual onde apenas o primeiro assunto do dia aparecia no calendário.

### Alterado
- Tecla ENTER agora confirma cadastros e atualizações de intervalos.
- Versão do Assembly atualizada para 1.1.0.
